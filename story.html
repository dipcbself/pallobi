The Flower

the story of the making

This is not a project in the usual sense.

It's something I built slowly, quietly — for someone who matters to me.

It's a digital portrait made only with HTML and CSS, inspired by a photo I love the most. Just thousands of tiny divs, placed one by one.

It's simple.
It's imperfect.
It's probably outdated.

But building it, brick by brick, gave me joy.

Why I Started This

One day, I thought:

"Instead of just looking at her photos, how about I try to build one?"

It sounded cool. Like laying bricks!

So I decided to turn a real photo into pixel art using only code.

Not because it's practical. Because it felt meaningful to me!

Planning

I didn't have any real plan.

It went like this:

  • "Can I really do this with divs?"
  • "This looks bad."
  • "Let me try again."
  • "Still bad."
  • "One more time."
  • "Oh… this is getting better."

Repeat.

How I Built It

Step 1: Basic Pixel Grid

First, I made a grid using CSS Grid.

Each cell = one pixel.

At first, it was blurry. But it was a start.

Step 2: Increasing Detail

Then I kept improving:

  • Grid size
  • Color accuracy
  • Shading
  • Contrast

Every upgrade made the file bigger and slower.

But also more alive.

I didn't stop when it was "good enough". I stopped when it felt right.

Step 3: Zoom & Interaction

I wanted people to:

  • Zoom in
  • See the pixels
  • Feel the effort

So I added scroll-based zoom and panning.

Step 4: Atmosphere

Later, I added:

  • Soft glow
  • Framing
  • Background

To make it feel more emotional and cinematic.

Step 5: Mobile Optimization

For mobile devices, I had to use canvas-style wrapping.

Otherwise, loading more than 346,000 divs and 178,000+ pixels could crash the browser.

Tech Stack

  • HTML
  • CSS
  • JavaScript
  • Some sleepless nights
  • A lot of patience

What This Means to Me

Compared to modern graphics, this is:

  • Low resolution
  • Heavy
  • Inefficient

I know!

And I'm okay with that.

Because that was the point! This wasn't about optimization. It was about intention. Every pixel is there because I wanted it there.

This project won't make me money. It didn't go viral. It won't change the world.

But it changed something in me.

It reminded me that:

I still know what real feelings look like.
I can still sit down, focus, and build something when I care.
I can choose the longer road when I want.

And I'm happy I did.

Was It Worth It?

Yes.

Because it's mine!
